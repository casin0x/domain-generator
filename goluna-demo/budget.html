How the budget breaks down: Total 3M SEK (~$285.7K) = 500K bankroll + 2.5M SEK (~$238K) operating. The operating budget funds the shared fixed cost base (salaries, Cubeia, hosting, security, tooling, one-time setup — the sections below) plus each path’s growth/marketing. The fixed base is ~the same across paths; the growth allocation differs — Path A ~$130K · Path B ~$127K · Path C $112.5K ($87.5K Anzil capped $100K + $25K clipping via a separate vendor; influencer activation cut, M2 kill checkpoint) — each a 3-month pilot (M2–M4) with a GO/KILL gate at M4. Allocation decision (2026-06-11): 1M SEK (~$95K) of the working budget is dedicated to Brazil + Anzil; after fixed obligations (~745K SEK M1–M6: salaries, platform, setup), the remaining ~755K SEK (~$72K) goes to higher-ARPU markets and channel experiments — the $25K clipping prepay draws from this pot, and the same pot doubles as the M7+ runway reserve.
